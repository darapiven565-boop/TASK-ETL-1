# Звіт про обробку датасету UK-500

## 1. Завантаження та первинний огляд
- Завантажено CSV-файл із UK-500.
- Переглянуто перші рядки, інформацію про типи даних, статистику.
- Перевірено пропуски, дублікати, список колонок.

## 2. Очищення даних
- Приведено до нижнього регістру `email` і `web`.
- Видалено зайві пробіли у `phone1` та `phone2`.
- Усі текстові колонки стандартизовано: прибрано зайві пробіли, вирівняно формат.
- Створено автоматичне визначення колонок email/web/phone/fax.

### Очищення телефонів
- Залишено лише цифри та можливий знак `+`.
- Порожні значення перетворено в `NaN`.

### Форматування назв
- Колонки з іменами, адресами й містами приведено до `Title Case`.

## 3. Нові колонки
- `full_name` — ім’я + прізвище.
- `domain` — домен email.
- `is_gmail` — мітка Gmail-користувачів.
- Створено фільтр компаній із LLC/Ltd.

## 4. Вибірки та аналітика
- Отримано вибірки: перші 10 рядків, кожен 10-й запис, 5 випадкових рядків.
- Пораховано:
  - ТОП-3 email-доменів (інші домени по одному разу)
  gmail - 130
  yahoo - 123
  hotmail - 114
  - в яких містах найбільше записів:
  Central Ward - 3
  City and Hunslet Ward - 3

## 5. Збереження результатів
Збереження даних
- очищений датафрейм → `data/uk500_clean.csv`
- користувачі gmail → `data/gmail_users.csv`
- Excel зі статистикою:
  - `domens` — топ домени
  - `Cities` — топ міст

## 6. Підсумок
Виконано повне очищення, підготовку та базовий аналіз даних. Топ 5 по містах не знайдено, бо данні занадто різні, але найбільші значення вказано. Набір даних приведено до стандартизованого та аналітично готового формату.

## 7. Найкорисніші трансформації
1. Стандартизація тексту (`standartize_text`) — прибирає зайві пробіли та уніфікує формат.
2. Очищення телефонів (`clean_phone`) — спрощує подальшу роботу з номерами.
3. Title Case для імен, адрес та міст — покращує читаність і подальший аналіз.
4. Створення колонок `full_name`, `domain`, `is_gmail` — корисно для сегментації та агрегації.
5. Фільтр компаній LLC/Ltd — дозволяє швидко виділити юридичні осіб.

## 8. Висновки та цікаві спостереження
- Найпопулярніші email-домени: gmail.com, yahoo.com, hotmail.com.
- Виявлено невелику кількість пропусків і дублікатів, що легко очистити.
- Gmail-користувачі становлять помітну частину датасету.

## 9. Можливості покращення у майбутньому
1. Більш глибока очистка текстових даних (наприклад, видалення зайвих символів в адресах і компаніях).
2. Валідація email та веб-адрес через регулярні вирази.
3. Додатковий аналіз телефонів за країнами (з використанням коду +).
4. Візуалізація топових міст та доменів для більш наочного аналізу.
5. Автоматизація процесу обробки для великих датасетів.
